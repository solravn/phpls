
1. Создать таблицу счетчиков tbl_counter (./init.sh migrations generate или (внутри контейнера) /bin/huston generate)
Структура:
    name TEXT NOT NULL PRIMARY KEY (имя счетчика)
    value INT NOT NULL (значение счетчика)

2. Создать модель Counter для таблицы выше
Она должна уметь отдавать и обновлять счетчик по имени.

3. Создать контроллер с action'ами:
- 1. phpls.loc/counter/view/$counterName
Который будет просто выводить на странице текущий счетчик по имени $counterName.
Например "Счетчик blabla: 3".
Если такой счетчик ни разу не вызывался, то выводит 0.
- 2. phpls.loc/counter/increase/$counterName
Который должен увеличить счетчик по имени $counterName в базе данных на + 1.
Если счетчика еще нет, то нужно создать и записать 1.
Если счетчик успешно увеличен, пишем на странице "ok".
